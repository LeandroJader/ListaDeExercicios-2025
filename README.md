# ListaDeExercicios-2025

	
# projeto

Desenvolvido durante o curso Fullstack da  [Academia do Programador](https://www.academiadoprogramador.net) 2025


# detalhes

este projeto conta com cerca de 23 exercicios, cada um deles realiza uma certa tarefa em espeifico


# funcionalidades 
* Exercicio-01: Calcular o volume de uma caixa retangular.

* Exercicio-02: Calcular o volume de um cilindro.

* Exercicio-03: Calcular a média de combustível.

* Exercicio-04: Converter graus Celsius para Fahrenheit.

* Exercicio-05: Acrescentar comissão ao salário base.

* Exercicio-06: Calcular a média harmônica.

* Exercicio-07: Calcular a média ponderada.

* Exercicio-08: Verificar se o número é primo.

* Exercicio-09: Ler as dimensões de um terreno.

* Exercicio-10: Calcular a quantidade arrecadada por uma padaria.

* Exercicio-11: Calcular quantos dias uma pessoa já viveu.

* Exercicio-12: Calcular o salário final, acrescentando aumento e descontos.

* Exercicio-13: Verificar se a soma de A + B é menor que C.

* Exercicio-14: Exibir os números em ordem decrescente.

* Exercicio-15: Calcular o índice de massa muscular.

* Exercicio-16: Verificar se o número é par ou ímpar.

* Exercicio-17: Ler valores A e B e realizar soma ou multiplicação de acordo com a escolha.

* Exercicio-18: Imprimir números ímpares entre 100 e 200.

* Exercicio-19: Somar os números ímpares que são múltiplos de três.

* Exercicio-20: Ver a tabuada de um número.

* Exercicio-21: Calcular o fatorial de um número.

* Exercicio-22: Calcular a sequência de Fibonacci.

* Exercicio-23: FizzBuzz (exercicio de múltiplos de 3 e 5).

## Requisitos

- .NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.
---
## Como Usar

#### Clone o Repositório
```
https://github.com/LeandroJader/ListaDeExercicios-2025
```

#### Navegue até a pasta raiz da solução
```
ListaDeExercicios-2025
```

#### Restaure as dependências
```
dotnet restore
```

#### Navegue até a pasta do projeto
```
ListaDeExercicios-2025
```

#### Execute o projeto
```
dotnet run
```
